Mask Pattern Guide

Valid characters:

C - uppercase consonant

c - lowercase consonant

V - uppercase vowel

v - lowercase vowel

# - random digit (0-9)

% - escape next character (treat as literal)

3n - repeat next character

Examples:

  • "Cvcvc#" → "Susan5"
  • "3#" → "777"
  • "2V3#" → "AA777"
  • "4%_" → "____"
